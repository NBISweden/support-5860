Shiny app

The shiny app can be installed as an R package.

# install dependencies
install.packages(c("shiny","DT","readxl","shinythemes","remotes"),
repos="https://cloud.r-project.org/")

# install package
remotes::install_github("NBISweden/support5860")

# load package
library(support5860)
runShiny()
